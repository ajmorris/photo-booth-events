A block-based WordPress plugin for virtual photo booth functionality with camera capture, frame overlays, and gallery display.

== Description ==

Virtual Photo Booth is a modern WordPress plugin that allows website visitors to:

* Capture photos using their device camera
* Apply optional frame overlays
* Upload photos to WordPress
* View photos in a live gallery
* Admin moderation of uploaded photos

The plugin uses Gutenberg blocks for easy integration and follows WordPress coding standards.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/virtual-photo-booth` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > Virtual Photo Booth to configure default settings
4. Create a Photo Booth Event in the admin
5. Add the Photo Booth and Gallery blocks to your pages

== Frequently Asked Questions ==

= Does this work on mobile devices? =

Yes, the plugin uses the device camera API which works on modern mobile browsers.

= Can I customize the frame images? =

Yes, you can set frame images per event or use a default frame in the settings.

= How do I moderate photos? =

Navigate to Photo Booth Events > Moderate Photos in the admin menu.
